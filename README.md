# FUTURE_CS_01
Vulnerability Assessment Report (VAPT)

Project Overview
This project documents the results of a Vulnerability Assessment conducted on a live test website- preview.owasp-juice.shop. The assessment focused on identifying and analysing common web application security vulnerabilities using industry-standard tools.

Objective
The main objective of this task was to:
- Identify potential security weaknesses in a web application 
- Classify vulnerabilities based on severity 
- Provide clear and simple remediation recommendations
  
Tools Used
- Nmap 
- OWASP ZAP (Passive Scan) 
- Browser Developer Tools 
- Canva (Report design) 

Summary of Work
- Performed vulnerability assessment on a live website 
- Identified and documented security weaknesses using automated and manual techniques 
- Classified findings based on severity (Low / Medium / High/ Informational) 
- Produced a structured vulnerability assessment report with remediation suggestions 

 Deliverables
- Vulnerability Assessment Report (PDF via Canva) 
- GitHub documentation of findings 

Disclaimer
The project VAPT represents a point-in-time analysis, and new vulnerabilities may emerge over time.
